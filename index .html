Final Project Proposal

Student Tourism Agency Web Platform

Our final MIS-455 project is a student-centered tourism web application that supports browsing, booking, and managing travel packages designed exclusively around university semester breaks.

Project Objective

To develop a dynamic, student-focused e-commerce website specializing in budget-friendly travel, serving as a centralized platform for browsing, booking, and managing domestic and international travel packages exclusively scheduled during academic semester breaks.

Key Features & Components

  • Package Presentation: Visually appealing display of travel packages, clearly categorized as Domestic and International.
  • Budget & Scheduling Focus: Transparent, student-friendly pricing and travel dates aligned with academic break periods.
  • Online Booking: Inquiry-based booking flow that lets students initiate bookings directly from the website.
  • Direct Email Contact: “Send Message” contact form integrated with EmailJS for fast inquiry handling.
  • Responsive Design: Optimized viewing and usability across mobile, tablet, and desktop devices.

Technical Implementation

Frontend

Built using HTML5, CSS, and JavaScript to ensure a clean, maintainable, and responsive user interface.

Backend / Communication

Integrated with EmailJS (JavaScript SDK) so that booking and contact forms send messages directly from the website to the agency’s email address.
